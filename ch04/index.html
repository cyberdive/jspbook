Using Beans

This chapter introduced beans. Beans are Java's standard component model, and they integrate very well with JSPs. Beans help separate Java code from HTML by providing standard tags which allow the JSP to get data to and from the bean, via the bean's properties. Beans also make writing dynamic pages that use forms easier, by providing easy ways to send form data into beans, and get results out.
