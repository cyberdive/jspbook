Java News Today

This chapter took the Java News Today site from a collection of partially-functional pages to a full, usable site. The keys to doing this was a database that provided a place to store the data, and a number of beans that make this data available to JSP pages.

  • The JNT home page. Note: due to a bug in the current implementation of Tomcat, the quiz does not work in this version of the site. (postgresql)
  • The JNT newsdesk, which allows new stories to be created. Log in as "Sifl", password "sock" to start entering new stories. (postgresql)
