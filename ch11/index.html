Java News Today: Personalization

This chapter added personalization to the Java News Today site. Users can now personalize a number of aspects of the site manually, and in addition the site will track which articles each users reads, and will adjust itself automatically to highlite other articles that may be of interest.

  • The JNT home page. Note: due to a bug in the current implementation of Tomcat, the quiz does not work in this version of the site. (postgresql)
