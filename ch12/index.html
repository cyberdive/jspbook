Java News Today: Ads and ecommerce

This chapter added advertisements and electronic commerce features to the Java News Today site. Using a system similar to personalization, ads are tailored to user interests.

  • The JNT home page. Note: due to a bug in the current implementation of Tomcat, the quiz does not work in this version of the site. (postgresql)
  • The ad-creation page. Note that each keyword has a value assocated with it. These values will be used to match the ads by weight to user interests. (postgresql)
  • Th JNT catalog. Try selecting a few items, then checking out. (postgresql)
