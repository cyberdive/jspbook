Advanced techniques

This chapter surveyed some advanced JSP techniques. Although many of these are unlikely to be used on a day-to-day basis, there are certainly times when they may come in handy.

  • A JSP that shows a list of prime numbers, using a declared method. (source).
  • Another JSP that shows a list of prime numbers, this time pre-populating a list using JspInit. (source).
  • Yet another list of primes, built by extending the base JSP class. (JSP source -- base class source)
  • A JSP that builds an XML page containing information about a CD. This may cause your borower to pop-up a window asking you to save the file. (source).
  • Another version of the XML JSP, this time the JSP is itself a well-formed XML document. This does not completely work yet, since the XML versions of some JSP tags are not yet implemented in Tomcat. (source).
