Java Server Pages: Dynamic Web Sites Made Easy

Included on this CD-ROM are all the code examples from the book. From this page you can select any chapter and view the JSP text and bean code from that chapter. Even more exciting, however, all the code can actually be run!

In order to do this, you must first set up and run Tomcat, the JSP engine from the Jakarta project. The latest version of Tomcat at the time this book was written is included on this CD. Instructions for setting everyting up are included here. You should also periodically check the Jakarta home page for new versions and related projects.

Many of the examples in this book rely on a database. We used PostgreSQL, a high-quality free database. PostgreSQL is available from here, and instructions for setting it up are included on this CD. Alternately, you might prefer to use MySQL or Msql. If so, some the SQL in the examples will need to be modified.

Examples that require PostgreSQL will be indicated with the word (postgresql) in parenthesis.

Also included on this CD are two utilities from Canetoad Software. Table2Bean creates beans from databse tables. BeanEditor allows users to edit and save serializable beans.

Code is available for the following chapters. Due to a bug in the current version of Tomcat, it may be necessary to shut down and restart it between visits to each chapter.

  • Simple JSPs
  • Request-time expressions
  • Using Beans
  • More advanced JSPs
  • Bean scopes
  • Writing beans
  • Databases
  • Java News Today
  • Personalization
  • Dynamic ads and e-commerce
  • Servlets
  • Advanced Topics
